# 🍺 Beer Guide API
Este projeto é um Guia de Cervejas Artesanais desenvolvido para a disciplina de Java Advanced. A aplicação consiste em um sistema de gerenciamento de cervejarias e seus respectivos rótulos de cerveja, utilizando boas práticas de desenvolvimento como Cache, Documentação Automática e Monitoramento. 

# 📖 Documentação da API (Swagger)
A documentação interativa da API, onde é possível visualizar todos os endpoints, modelos e realizar testes em tempo real, está disponível no link abaixo com a aplicação em execução:

👉 [http://localhost:8080/swagger-ui/index.html ](http://localhost:8080/swagger-ui/index.html#/)
